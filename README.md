# Ameren iCenter External Website

External [website](https://amerenicenter.github.io/) for the Ameren Innovation Center at UIUC's Research Park.

### Run Locally

```
npm install
npm run start
localhost:3000
```

### Deployment

This website is deployed on GitHub Pages.

```
npm run deploy
```
